## 🎯 User Flow

### Learning Path
```
1. Hero → Mulai Belajar
2. Pengertian → Baca definisi
3. Unsur-Unsur → Klik diagram
4. Nilai Phi → Pahami konstanta
5. Keliling & Luas → Coba kalkulator
6. Materi SMK → Pilih topik
7. Latihan Soal → Kerjakan soal
8. AI Chat → Tanya jika bingung
```

### Calculator Flow
```
1. Input nilai
2. Lihat formula
3. Hasil muncul real-time
4. Ubah nilai → hasil update
5. Coba nilai lain
```

### Problem Solving Flow
```
1. Baca soal
2. Coba kerjakan sendiri
3. Klik accordion
4. Lihat langkah-langkah
5. Pahami solusi
6. Coba soal berikutnya
```
